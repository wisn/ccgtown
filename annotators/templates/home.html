CCGtown

Annotating CCG lexicons couldn't be this easy.
Input sentence, annotate it, generate the derivation, and viola!

Try It Now

Features

Multi-project

You need to do both wide-range and domain-specific projects at the same time? No problem. We got you covered. Create as many projects as you want.

Auto-assign Annotation

Tired annotating the same words again and again? Let the computer do it for you! CCGtown will annotate the same words that you have annotated before.

Generate CCG Derivation

With the help of NLTK, we will generate the CCG derivation for you. Well, as long as you provide the right annotation. That's it.

Editable CCG Derivation

You want to derive the CCG by yourself? Or, does the generated CCG derivation doesn't seems right to you? Feel free to edit the CCG derivation!

Export to JSON

Export your work to JSON and use it elsewhere. Although the current exported format is not that great, we are working on it!

100% Open Source

CCGtown is an open source project. Meaning that you are free to clone and extend this project as you wish. This is a maintained project so the last commit won't be a year ago.
